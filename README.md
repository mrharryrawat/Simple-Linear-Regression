# Simple-Linear-Regression

## About This Project

This project was developed as part of my Machine Learning learning journey. It demonstrates how a Simple Linear Regression model can be trained and deployed through a Flask web application to predict salaries based on years of experience.

The project focuses on understanding the end-to-end workflow of a machine learning application, including data preparation, model training, model serialization, and web deployment.

To deploy it I used AI-assistance especially in front-end

## Data

This project uses a dataset containing Years of Experience and Salary data. A Linear Regression model is trained on this dataset to predict salary based on the number of years of experience.

**Note:** This is a beginner-level educational project built on a small dataset and is not intended for real-world salary prediction.
